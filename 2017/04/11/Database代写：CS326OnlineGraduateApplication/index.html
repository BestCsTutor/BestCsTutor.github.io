Database代写：CS326OnlineGraduateApplication

Database代写
发布日期:   2017-04-11

根据题目要求，代写完成一个系统的数据库设计，主要就是用Visio绘制E-R图。
和其他的E-R图作业不同，这个作业还增加了属性的约束条件，增加了设计的难度。

Description

For Homework 1, you will create an Entity Relationship Diagram (ERD) for the
online Graduate Application (details included below). Be sure to include the
following:

  1. Draw an Entity Relationship Diagram (ERD).
    * Entity names must follow naming guidelines.
    * The ERD must be normalized to 3NF.
  2. Include both ordinality (i.e. required or optional) and cardinality (i.e. 1:1, 1:M, M:N, etc).
  3. Include the relationship narratives between the entities.
  4. Identify primary keys for each entity and include all appropriate foreign keys.
  5. Use Visio to complete the drawing.
